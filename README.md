
# Football SP

O Football SP é um site com o objetivo de explorar a rica história do futebol em São Paulo, com um resumo das origens e momentos memoráveis da história e times que o moldaram. Desde sua introdução por Charles Miller até o estabelecimento de alguns clubes icônicos.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- Figma
- Font Awesome
- Svg Repo

## Funcionalidades
- Imagens interativas
- Perfis de clubes icônicos
- Galeria de imagens
- Seção de Quiz
